Introduction

D'où vient l'intégrale ?
Rappel — Intégrale indéfinie

$F$ est une primitive de $f$ si $F' = f$. On note $\int f(x)\,dx$ la primitive de $f$ (unique à une constante près). Dans ce chapitre : les intégrales définies.

Motivation — Accumulation

Vitesse constante $v=100$ km/h pendant 3h → distance $= 100\times3 = 300$ km = aire du rectangle sous la courbe.
Vitesse variable $v(t)$ → distance $=$ aire sous la courbe $= \int_a^b v(t)\,dt$.

Définition informelle — Intégrale définie
Soit $f:[a,b]\to\mathbb{R}$. On note $\displaystyle\int_a^b f(t)\,dt$ l'aire orientée entre le graphe de $f$, l'axe $x$ et les droites $x=a$, $x=b$.
Aire au-dessus de l'axe $x$ : positive. Aire en dessous : négative.

Convention : $\displaystyle\int_b^a f(t)\,dt = -\int_a^b f(t)\,dt$.
Aire orientée (bleu = positif, rouge = négatif)
Fonction : ∫ ≈ —

Théorème Fondamental de l'Analyse (TFA)

Par cœur — Q5 = 0/3 à l'exam raté
Par cœur — version 1
TFA — Version 1
Soit $f : [a,b] \to \mathbb{R}$ une fonction continue et définissons $F : [a,b] \to \mathbb{R}$ : $$F(x) = \int_a^x f(t)\,dt$$ Alors $F$ est dérivable et $F' = f$.
Ce que ça dit

La dérivée de "l'intégrale de $f$ de $a$ à $x$" redonne $f(x)$. Conséquence : toute fonction continue admet une primitive. Dérivation et intégration sont inverses l'une de l'autre.

Par cœur — version 2
TFA — Version 2
Soit $f : [a,b] \to \mathbb{R}$ une fonction continue et $F$ une primitive de $f$. Alors : $$\int_a^b f(t)\,dt = F(b) - F(a) = \big[F(t)\big]_a^b$$
Preuve que v1 implique v2 (à lire + comprendre, pas reproduire mot pour mot)

Pose $G(x) = F(x) - \int_a^x f(t)\,dt$. Par TFA v1 : $G'(x) = f(x)-f(x)=0$. Donc $G$ est constante. En $x=a$ : $G(a)=F(a)$. En $x=b$ : $G(b)=F(a)$, i.e. $F(b)-\int_a^b f=F(a)$, d'où $\int_a^b f = F(b)-F(a)$. $\square$

Calcul avec TFA v2

Calculons $\displaystyle\int_{-1}^1 (-t^2+1)\,dt$.

Style prof
Primitive : $F(t) = -\dfrac{t^3}{3}+t$. Par TFA (v2), $f$ étant continue :
$$\int_{-1}^1(-t^2+1)\,dt = \left[-\frac{t^3}{3}+t\right]_{-1}^1 = \left(-\frac{1}{3}+1\right)-\left(\frac{1}{3}-1\right) = \frac{2}{3}+\frac{2}{3} = \frac{4}{3}$$
Application TFA v1 — Dériver une intégrale
$\dfrac{d}{dx}\displaystyle\int_5^x (f(t))^2\,dt$ : pose $g(t)=(f(t))^2$ continue. Par TFA v1 : $= g(x) = (f(x))^2$.

Définition formelle — Partitions & Riemann

La vraie définition rigoureuse
Idée

Approcher l'aire par des rectangles (sommes de Riemann), puis passer à la limite. Si les approximations par défaut et par excès convergent vers la même valeur → c'est l'intégrale.

Définition — Partition
Une partition de $[a,b]$ est un ensemble $\{x_0,\ldots,x_n\}$ avec $x_0=a < x_1 < \cdots < x_n = b$.
Définition — Sommes $I_P(f)$ et $S_P(f)$
Soit $P=\{x_0,\ldots,x_n\}$ une partition de $[a,b]$. On pose : $$h_i = \inf_{x\in[x_{i-1},x_i]} f(x), \qquad H_i = \sup_{x\in[x_{i-1},x_i]} f(x)$$ $$I_P(f) = \sum_{i=1}^n h_i(x_i-x_{i-1}) \quad\text{(somme inférieure)}, \qquad S_P(f) = \sum_{i=1}^n H_i(x_i-x_{i-1}) \quad\text{(somme supérieure)}$$ On a toujours : $I_P(f) \leq \displaystyle\int_a^b f(x)\,dx \leq S_P(f)$.
Lemme — Partitions emboîtées
Si $P_1 \subseteq P_2$ (i.e. $P_2$ raffine $P_1$), alors : $$I_{P_1}(f) \leq I_{P_2}(f) \qquad \text{et} \qquad S_{P_1}(f) \geq S_{P_2}(f)$$

Ajouter des points à une partition améliore les deux approximations.

Sommes de Riemann pour $f(x)=x^2$ sur $[0,1]$
$n$ rectangles : n = 5
$I_P = $ — $S_P = $ — Exacte : 1/3 ≈ 0.333
Définition — Intégrale de Riemann
On définit : $$\mathcal{I}(f) = \sup\{I_P(f) \mid P\text{ partition}\}, \qquad \mathcal{S}(f) = \inf\{S_P(f) \mid P\text{ partition}\}$$ $f$ est intégrable au sens de Riemann sur $[a,b]$ si $\mathcal{I}(f) = \mathcal{S}(f)$. Dans ce cas : $$\int_a^b f(x)\,dx = \mathcal{I}(f) = \mathcal{S}(f)$$
Critère de Riemann
$f:[a,b]\to\mathbb{R}$ bornée est intégrable $\iff$ $\forall\varepsilon>0$, $\exists$ partition $P_\varepsilon$ de $[a,b]$ telle que : $$S_{P_\varepsilon}(f) - I_{P_\varepsilon}(f) < \varepsilon$$
Exemple — $f(x)=x$ sur $[0,1]$ est intégrable (preuve)
Partition uniforme $P_n = \{0,\frac{1}{n},\ldots,1\}$. Sur $[\frac{i-1}{n},\frac{i}{n}]$ : $h_i=\frac{i-1}{n}$, $H_i=\frac{i}{n}$.
$I_{P_n} = \frac{n-1}{2n}$, $S_{P_n} = \frac{n+1}{2n}$, donc $S_{P_n}-I_{P_n} = \frac{1}{n}$.
Pour $\varepsilon>0$, prendre $n>\frac{1}{\varepsilon}$ : le critère est satisfait. Donc $f$ est intégrable et $\int_0^1 x\,dx = \frac{1}{2}$. $\square$
Proposition — Continue ⟹ Intégrable
Si $f:[a,b]\to\mathbb{R}$ est continue, alors $f$ est intégrable au sens de Riemann.

Cela inclut : polynômes, exponentielles, trigo, et toute combinaison algébrique de ces fonctions.

Pas toute fonction est intégrable !
Fonction de Dirichlet : $f(x) = \mathbb{1}_\mathbb{Q}(x) = \begin{cases}1 & x\in\mathbb{Q} \\ 0 & \text{sinon}\end{cases}$. Pour toute partition : $I_P=0$ et $S_P=1$ ($\mathbb{Q}$ et $\mathbb{R}\setminus\mathbb{Q}$ sont denses dans $\mathbb{R}$). Donc $\mathcal{I}=0\neq1=\mathcal{S}$ → non intégrable.

Propriétés de l'intégrale définie

Les règles de calcul
Proposition 1 — Linéarité & Additivité
Soient $f,g:[a,b]\to\mathbb{R}$ intégrables.

1. Linéarité : $\displaystyle\int_a^b(c_1f+c_2g) = c_1\int_a^b f + c_2\int_a^b g$

2. Additivité : $\forall c\in]a,b[$ : $\displaystyle\int_a^b f = \int_a^c f + \int_c^b f$
Proposition 2 — Inégalités
1. Si $f\leq g$ sur $[a,b]$ : $\displaystyle\int_a^b f \leq \int_a^b g$

2. Si $|f|$ intégrable : $\displaystyle\left|\int_a^b f\right| \leq \int_a^b |f|$
Proposition 3 — Fonctions paires / impaires (gain de temps !)
Soit $f:[-a,a]\to\mathbb{R}$ intégrable.

Si $f$ impaire ($f(-x)=-f(x)$) : $\displaystyle\int_{-a}^a f(x)\,dx = 0$

Si $f$ paire ($f(-x)=f(x)$) : $\displaystyle\int_{-a}^a f(x)\,dx = 2\int_0^a f(x)\,dx$
Exemple
$\int_{-1}^1 x^3\,dx = 0$ ($x^3$ impaire)
Exemple
$\int_{-2}^2 x^2\,dx = 2\int_0^2 x^2\,dx = \frac{16}{3}$ ($x^2$ paire)

Techniques d'intégration

IPP et Substitution
Intégration par Parties (IPP)
Soient $f,g:[a,b]\to\mathbb{R}$ dérivables, $f'$, $g'$ continues. Alors : $$\int_a^b f(x)\,g'(x)\,dx = \big[f(x)\,g(x)\big]_a^b - \int_a^b f'(x)\,g(x)\,dx$$
Règle LIATE — choisir $f$

Prendre $f$ = la fonction la plus à gauche : Logarithme > Inverse-trigo > Algèbre (polynôme) > Trigo > Exponentielle.

IPP — $\int_0^1 xe^x\,dx$
Style prof
$f(x)=x$, $g'(x)=e^x$ → $f'(x)=1$, $g(x)=e^x$.
$$\int_0^1 xe^x\,dx = \big[xe^x\big]_0^1 - \int_0^1 e^x\,dx = e - \big[e^x\big]_0^1 = e-(e-1) = \boxed{1}$$
Intégration par Substitution
Soit $f$ continue et $g$ dérivable avec $g'$ continue. Si $g(\alpha)=a$ et $g(\beta)=b$, alors : $$\int_a^b f(x)\,dx = \int_\alpha^\beta f(g(t))\,g'(t)\,dt$$

Pratique : $u=g(t)$, $du=g'(t)\,dt$, puis changer les bornes.

Toujours changer les bornes !
Sur une intégrale définie, quand tu poses $u=g(x)$ : $x=a \to u=g(a)$ et $x=b \to u=g(b)$. Pas besoin de revenir en $x$.
Subst. — $\int_0^{\sqrt{e-1}}\frac{4x}{1+x^2}\,dx$
Pose $u=1+x^2$, $du=2x\,dx$. Bornes : $u: 1\to e$.
$=\int_1^e\frac{2}{u}\,du=[2\ln u]_1^e=\boxed{2}$
Subst. — $\int_0^2 te^{-t^2}\,dt$
Pose $u=t^2$, $du=2t\,dt$. Bornes : $u:0\to4$.
$=\frac{1}{2}\int_0^4e^{-u}\,du=\frac{1}{2}(1-e^{-4})=\boxed{\frac{1-e^{-4}}{2}}$

Intégrales impropres

Bornes infinies ou fonction qui explose
Pourquoi "impropres" ?

Les intégrales définies supposent $f$ définie et bornée sur $[a,b]$ fini. Une intégrale est impropre si (1) une borne est $\pm\infty$, ou (2) $f$ explose en un point. Dans les deux cas : remplacer la borne problématique par une limite.

Définition — Domaine borné (fonction explose en une borne)
Soit $f:[a,b[\to\mathbb{R}$ continue. Si $\displaystyle\lim_{u\to b^-}\int_a^u f(x)\,dx$ existe, on la dénote $\int_a^b f(x)\,dx$.
Si la limite est réelle → converge. Sinon → diverge.
De façon analogue pour $f:]a,b]\to\mathbb{R}$.
Définition — Domaine non borné
Soit $f:[a,+\infty[\to\mathbb{R}$ continue et bornée. Si $\displaystyle\lim_{b\to+\infty}\int_a^b f(x)\,dx$ existe, on la dénote $\int_a^{+\infty}f(x)\,dx$.
Pour $f:\mathbb{R}\to\mathbb{R}$ : $\displaystyle\int_{-\infty}^{+\infty}f := \int_{-\infty}^0 f + \int_0^{+\infty}f$.
$\int_0^1\frac{1}{\sqrt{x}}\,dx$ (explose en 0)
$:=\lim_{u\to0^+}[2\sqrt{x}]_u^1=2-0=\boxed{2}$. Converge.
$\int_3^4\frac{1}{x-3}\,dx$ (explose en 3)
$:=\lim_{u\to3^+}[\ln|x-3|]_u^4=+\infty$. Diverge.
$\int_1^\infty\frac{1}{x^3}\,dx$
$:=\lim_{b\to\infty}\left[-\frac{1}{2x^2}\right]_1^b=\frac{1}{2}$. Converge.
$\int_1^\infty\frac{1}{x}\,dx$
$:=\lim_{b\to\infty}[\ln x]_1^b=+\infty$. Diverge.
Critère pratique — $\int_1^\infty x^{-p}\,dx$
$p>1$$p=1$$p\leq1$
$\frac{1}{p-1}$ converge$+\infty$ diverge$+\infty$ diverge

Analogue pour $\int_0^1 x^{-p}\,dx$ : converge si $p<1$, diverge si $p\geq1$.

Longueur d'une courbe paramétrée

Mesurer la longueur d'un chemin
Définition — Longueur d'une courbe paramétrée
Si $\gamma:[a,b]\to\mathbb{R}^m$ est une courbe paramétrée dérivable, sa longueur est : $$L(\gamma) = \int_a^b \|\gamma'(t)\|\,dt \quad\text{où}\quad \|\gamma'(t)\| = \sqrt{\sum_{i=1}^m (\gamma_i'(t))^2}$$
Justification (slides)

Partition $\{0,\frac{1}{n},\ldots,1\}$ → longueur $\approx\sum_{i=1}^n\|\gamma(\frac{i}{n})-\gamma(\frac{i-1}{n})\| = \sum_i\frac{\|\gamma(\frac{i}{n})-\gamma(\frac{i-1}{n})\|}{\frac{1}{n}}\cdot\frac{1}{n} \xrightarrow{n\to\infty} \int_0^1\|\gamma'(t)\|\,dt$.

Arc de cercle sur $[0,\frac{3\pi}{2}]$
$\gamma(t)=(\cos t,\sin t)$, $\gamma'=(-\sin t,\cos t)$, $\|\gamma'\|=1$.
$L=\int_0^{3\pi/2}1\,dt=\frac{3\pi}{2}$.
Droite $\gamma(t)=(3t-1,2t)$ sur $[6,7]$
$\gamma'=(3,2)$, $\|\gamma'\|=\sqrt{13}$.
$L=\int_6^7\sqrt{13}\,dt=\sqrt{13}$.
TP 6.20 — Courbe $y^2=x^3$ de $(1,-1)$ à $(1,1)$
Style prof
Paramétrisation : $\gamma(t)=(t^2,t^3)$, $t\in[-1,1]$. (Vérif : $(\gamma_2)^2=t^6=(t^2)^3=(\gamma_1)^3$ ok, $\gamma(\pm1)=(1,\pm1)$ ok)
$\gamma'(t)=(2t,3t^2)$, $\|\gamma'(t)\|=|t|\sqrt{4+9t^2}$.
$$L=2\int_0^1 t\sqrt{4+9t^2}\,dt \xrightarrow{u=4+9t^2} \frac{2}{18}\int_4^{13}\sqrt{u}\,du = \frac{2}{27}\left[u^{3/2}\right]_4^{13} = \boxed{\dfrac{26\sqrt{13}-16}{27}}$$

Intégrales multivariées

Directement testé — Q4 exam
Généralisation

En 2D : $\iint_D f(x,y)\,dA$ = volume sous la surface. Cas particulier : $\iint_D 1\,dA$ = aire du domaine $D$. Difficulté principale : trouver les bonnes bornes selon la forme de $D$.

Définition — Pavé
Un pavé dans $\mathbb{R}^n$ est $R = [a_1,b_1]\times\cdots\times[a_n,b_n]$. Pour $n=2$ : un rectangle.
Définition — Intégrale sur un pavé (récurrence sur $n$)
• Si $n=1$ : $\int_R f = \int_a^b f(x)\,dx$.

• Si $n\geq2$ : $R=R'\times[a_n,b_n]$ ($R'=(n-1)$-pavé), et : $$\int_R f\,dV := \int_{a_n}^{b_n}\!\!\left(\int_{R'} f(x_1,\ldots,x_n)\,dV_{n-1}\right)\!dx_n$$ pourvu que toutes les intégrales itérées existent.
Théorème de Fubini
Soit $R=[a_1,b_1]\times[a_2,b_2]\subset\mathbb{R}^2$ et $f:R\to\mathbb{R}$ continue. Alors : $$\int_{a_1}^{b_1}\!\!\left(\int_{a_2}^{b_2}\!\! f(x,y)\,dy\right)\!dx = \int_{a_2}^{b_2}\!\!\left(\int_{a_1}^{b_1}\!\! f(x,y)\,dx\right)\!dy$$ Plus généralement dans $\mathbb{R}^n$ : pour $f$ continue, l'ordre d'intégration est interchangeable.
Fubini — $\iint_R(x^2+y)\,dA$, $R=[0,1]\times[0,2]$
$\int_0^2\!\int_0^1(x^2+y)\,dx\,dy = \int_0^2\left[\frac{x^3}{3}+yx\right]_0^1\!dy = \int_0^2\!\!\left(\frac{1}{3}+y\right)dy = \left[\frac{y}{3}+\frac{y^2}{2}\right]_0^2 = \frac{2}{3}+2 = \frac{8}{3}$
Définition — Intégrale sur un domaine borné quelconque
Soit $D\subset\mathbb{R}^n$ borné et $R$ un pavé contenant $D$. On pose : $$\tilde{f}(x,y) = \begin{cases}f(x,y) & (x,y)\in D \\ 0 & (x,y)\in R\setminus D\end{cases}$$ Alors : $\displaystyle\int_D f\,dV := \int_R\tilde{f}\,dV$ (si cette intégrale existe). Indépendant du pavé choisi.
Définition — Régions simples de $\mathbb{R}^2$
$D$ est verticalement simple si : $$D = \{(x,y)\in\mathbb{R}^2 \mid a\leq x\leq b,\; g_1(x)\leq y\leq g_2(x)\} \quad g_1,g_2\text{ continues}$$ $D$ est horizontalement simple si : $$D = \{(x,y)\in\mathbb{R}^2 \mid c\leq y\leq d,\; h_1(y)\leq x\leq h_2(y)\} \quad h_1,h_2\text{ continues}$$ On peut étendre cette définition à $\mathbb{R}^n$.
Proposition — Calcul sur domaine simple
Si $D$ est verticalement simple : $$\iint_D f(x,y)\,dA = \int_a^b\!\!\left(\int_{g_1(x)}^{g_2(x)}\!\! f(x,y)\,dy\right)\!dx$$ Si $D$ est horizontalement simple : $$\iint_D f(x,y)\,dA = \int_c^d\!\!\left(\int_{h_1(y)}^{h_2(y)}\!\! f(x,y)\,dx\right)\!dy$$

Style prof
Sur $[0,1]$, $x^3\leq x^2$ car $x^3=x^2\cdot x\leq x^2$ pour $x\leq1$. Domaine verticalement simple : $D=\{0\leq x\leq1,\;x^3\leq y\leq x^2\}$.
$$\iint_D 1\,dA = \int_0^1(x^2-x^3)\,dx = \left[\frac{x^3}{3}-\frac{x^4}{4}\right]_0^1 = \frac{1}{3}-\frac{1}{4} = \boxed{\dfrac{1}{12}}$$
Triangle (0,0)-(1,0)-(1,1) : $\iint_D(x+y)\,dA$
$D=\{0\leq x\leq1,\;0\leq y\leq x\}$.
$=\int_0^1\!\left[xy+\frac{y^2}{2}\right]_0^x\!dx=\int_0^1\frac{3x^2}{2}\,dx=\frac{1}{2}$
Triangle $x+y=1$ : $\iint_D(x+y)^2\,dA$
$D=\{0\leq y\leq1,\;0\leq x\leq1-y\}$.
$=\int_0^1\!\left[\frac{(x+y)^3}{3}\right]_0^{1-y}\!dy=\int_0^1\!\frac{1-y^3}{3}\,dy=\frac{1}{4}$

Propriétés de l'intégrale multiple
Soient $D\subset\mathbb{R}^n$ borné, $f,g:D\to\mathbb{R}$ intégrables.

1. Linéarité : $\int_D(c_1f+c_2g)=c_1\int_Df+c_2\int_Dg$

2. Additivité : Si $D=D_1\cup D_2$ avec $\text{Int}(D_1)\cap\text{Int}(D_2)=\emptyset$ : $\int_Df=\int_{D_1}f+\int_{D_2}f$

3. Monotonicité : $f\leq g$ sur $D\Rightarrow\int_Df\leq\int_Dg$

4. Si $|f|$ intégrable : $\left|\int_Df\right|\leq\int_D|f|$
